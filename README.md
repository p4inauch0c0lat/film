# film

Bot Discord pour fournir un embed de catalogue de streaming.

## Démarrage rapide

```bash
npm install
npm start
```

## Variables d'environnement

- `DISCORD_TOKEN` (obligatoire)
- `FILM_COUNT`
- `SERIES_COUNT`
- `SEASONS_COUNT`
- `EPISODES_COUNT`
- `TOTAL_LINKS`
- `STATUS_URL`

## Utilisation

Envoyez `!catalogue` dans un salon où le bot est présent pour obtenir l'embed principal.
